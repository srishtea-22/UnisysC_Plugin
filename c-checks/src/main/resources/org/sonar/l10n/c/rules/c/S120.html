Why is this an issue?

Shared naming conventions improve readability and allow teams to collaborate efficiently. This rule checks that all label names match a provided regular expression.

How to fix it

Rename labels with the expected naming convention

Code examples

Noncompliant code example

With the default regular expression ^[a-z][a-z0-9_]*$:

void foo() {
    INVALID_Label:  // Noncompliant
        printf("hello");
}

Compliant solution

void foo() {
    valid_label:
        printf("hello");
}
