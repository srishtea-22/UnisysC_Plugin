Why is this an issue?

This is required by IEC 61508, under good programming style.

Noncompliant code example

void func1() { /* Noncompliant - there are two points of exit */
  if (0) {
    return;
  }
}

int func2(int a) { /* Noncompliant - there are two points of exit */
  if (a > 0) {
    return 0;
  }
  return -1;
}

Compliant solution

void func1() {
  return;
}

void func2() {
}
